# Todo List Vue.js

This project is a personal practice project for learning the Vue 3 framework.  
It aims to help understand Vue basics including component structure, reactivity, event handling, and list rendering.

The project is built using Vue 3 + Vite.

---

## Branches Overview

### 1. main (default branch)
- Contains the initial version of the project
- Includes basic UI structure and CSS styling
- No core functionality implemented yet
- Suitable for beginners to start from scratch and practice adding features

### 2. feature/add-todo
- Contains the complete functional version
- Includes:
  - Adding todos
  - Deleting todos
  - Marking todos completed
  - Input validation (no empty items)
  - Reactive state and dynamic UI updates

---

## How to Use This Repository for Learning

Step 1: Clone the repository


Step 2: Switch to the `main` branch (recommended for learning)


Step 3: Install dependencies: `npm install`

Step 4: Run the project: `npm run dev`

Step 5: Open the local dev server: http://localhost:5173/
